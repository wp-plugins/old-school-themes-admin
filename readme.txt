Display links to the wp-admin appearance themes admin pages in both the dashboard and in the front-end toolbar.

== Description ==

Display links to the wp-admin appearance menu themes admin pages in both the dashboard and in the front-end toolbar. 

Includes links to the custom header and custom background pages that recently have disappeared from the dashboard appearance menu.

And in the front-end toolbar, display those links that recently have been replaced by links to the customizer. 

Except for Themes itself, assumes theme support is enabled for the various items : Widgets, Menus, Background, Header

== Installation ==

Download the plugin zip file

Upload via dashboard > plugins > add new > upload

Activate the plugin
